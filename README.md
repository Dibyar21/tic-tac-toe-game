An interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. Play against another player in the browser with real-time winner detection, draw handling, reset functionality, and an engaging user interface.

---

## 🚀 Features

- 🎮 Two-player gameplay (X vs O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset Game option
- 🆕 New Game functionality
- ⌨️ Typewriter animation for winner announcement
- 🎨 Clean and responsive UI
- ⚡ Instant game updates

## 🎯 How to Play

1. Player **O** starts the game.
2. Players take turns placing **O** and **X**.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all nine boxes are filled without a winner, the game ends in a draw.
5. Click **Reset Game** or **New Game** to start again.

---

## 🏆 Winning Conditions

A player wins by completing any one of the following:

- Horizontal row
- Vertical column
- Diagonal line
